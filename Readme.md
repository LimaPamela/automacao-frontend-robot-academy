* Instalar o Python
* Instalar o Robot
* Atualizar o Robot

* Instalar Library BuiltIn
* Instalar Library FakerLibrary    locale=pt_BR
* Instalar Library Screenshot
* Instalar Library SeleniumLibrary
* Instalar Library String
* Instalar Library strings_utils.py
